# RAG Ecommerce Analytics Assistant

An AI-powered ecommerce analytics assistant using Retrieval-Augmented Generation (RAG), semantic search, and LLM integration.

## Features

- PDF document ingestion
- Semantic search
- AI-generated answers
- Vector database retrieval
- Ecommerce analytics dashboard
- Knowledge chunk retrieval
- Streamlit web interface

## Technologies Used

- Python
- Streamlit
- Gemini API / OpenAI API
- FAISS / ChromaDB
- Pandas
- Plotly
